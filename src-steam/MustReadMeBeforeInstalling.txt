
** Needed ** You need to install the custom exe provided with the modpack.

If you are using non-installer version of modpack then all you have to do is copy paste the foler "World of Tanks Blitz" provided in the modpack and put it in ( C:\Program Files (x86)\Steam\steamapps\common\ )
 - and if it asks/promts you to overwrite/replace existing files then accept/agree the promt and after its finished then you are good to go and read to play!

To change the voice gender copy paste the corresponding ( Male or Female ) from CREW_M_F/(Female or Male/crewvoices.yaml into Blitz data folder ( C:\Program Files (x86)\Steam\steamapps\common\World of Tanks Blitz\Data\ )
